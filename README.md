# 😮What’s geuling?
- Google MLKit-TextRecognition 기술을 활용하여 이미지 속 글자를 인식합니다. 실시간으로 이미지 처리가 되기 때문에 불필요하게 이미지를 불러오거나 가져올 필요가 없습니다. 인식된 텍스트는 클립보드로 복사하거나 다른 환경을 통해 공유 할 수 있습니다.

- CameraPreviewer mode
- Take a picture and save
